# Unity AI - Explanation of files
## Assets
Contains all resources such as models, scripts, prefabs, etc.
### Assets / ML-Agents
Contains details about agent training using ML-Agents (Machine Learning Agents). Auto-generated.
### Assets / Materials
Contains components that define the visual appearance of objects in the game. 
### Assets / Prefabs
Contains preconfigured GameObjects that can be reused such as agents, props and training scenarios.
### Assets / Scenes
Contains scenes used for training agents.
### Assets / Scripts
Contains all scripts for agents, scenario-specific objects and utility.
## Packages
Contains all external libraries and plugins used in the project. Auto-generated.
## ProjectSettings
Contains a variety of essential configuration files, such as physics configuration or input configuration. Auto-generated.
